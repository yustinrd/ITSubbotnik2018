# Electron ITSubbotnik 2018 examples
Два примера продемонстрированных на ITSubbotnik 2018 в Саратове.


## Example 1
### Установка и запуск
1. `cd Ex1`
2. `npm i` or `yarn`  
3. `npm rebuild`
4. `npm start`

## Example 2
### Установка и запуск
В примере используется serialport. По умолчанию стоит порт COM4.
1. `cd Ex2`
2. `npm i` or `yarn`  
3. `npm rebuild`
4. `npm start`
